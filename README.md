# 📊 Employee Management System Database

## 📌 Overview

This project is a **MySQL-based Employee Management System** that manages employees, departments, leaves, and salary details. It is designed as a **practice + portfolio project** to demonstrate SQL skills including joins, subqueries, and real-world queries.

---

## 🛠️ Tech Stack

* MySQL
* SQL (DDL, DML, Joins, Subqueries)
* MySQL Workbench

---

## 📂 Project Structure

```
Employee-Management-System-DB/
│
├── schema.sql            # Database & table creation
├── data.sql              # Insert sample data
├── cleanup.sql           # Data cleaning queries
├── basic_queries.sql     # Beginner-level queries
├── advanced_queries.sql  # Advanced & real-world queries
└── README.md             # Project documentation
```

---

## 🗂️ Database Tables

### 🔹 Users

Stores login details and roles (admin / employee)

### 🔹 Employees

Contains employee personal and job-related information

### 🔹 Department

Stores department details

### 🔹 Leaves

Tracks employee leave records and status

### 🔹 Salary

Manages salary details including allowances and deductions
